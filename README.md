# Tour & Travel Website

A frontend web application built using Next.js.  
This project showcases different travel destinations with a clean and simple UI.

## Features
- Multiple destination pages (Delhi, Mumbai, Kerala, Goa, etc.)
- Responsive layout
- Beginner-friendly project structure
- Clean UI using CSS/SCSS

## Tech Stack
- Next.js
- React.js
- JavaScript
- CSS / SCSS

## Project Type
Frontend Only (No Backend / No Database)

## How to Run This Project Locally
1. Clone the repository  
2. Install dependencies  
   npm install  
3. Start the development server  
   npm run dev  
4. Open browser and visit  
   http://localhost:3000

## Purpose
This project is created for learning and academic purposes to understand frontend development using Next.js.
